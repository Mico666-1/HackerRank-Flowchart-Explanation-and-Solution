Read string s

 v
Compute prefix sums of character values (a=1, b=2, ...)
 |
 v
Read number of queries q
 |
 v
For each query:
 |
 v
Read l and r
 |
 v
Compute (prefix[r] - prefix[l-1]) mod 1000000007
 |
 v
Print result
 |
 v
End

The algorithm preprocesses the string by computing prefix sums of character values for efficient range queries. Each query retrieves the sum of values from position l to r in constant time using modular arithmetic.
