               │

│ Read n rows of integers into │

│ Initialize primarySum=0      │

               │

│ For i from 0 to n-1:               │
│  Add arr[i][i] to primarySum       │

               │
┌──────────────┴───────────────┐
│ Compute absolute difference  │
│   between primarySum and     │
│   secondarySum               │

               │
┌──────────────┴───────────────┐
│ Write result to output       │
└──────────────┬───────────────┘
               │
┌──────────────┴───────────────┐

This program calculates the absolute difference between the sums of the two diagonals in a square matrix. The diagonalDifference function sums the primary diagonal and the secondary diagonal elements, then returns their absolute difference.
