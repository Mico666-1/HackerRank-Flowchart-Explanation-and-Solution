       │

       │
┌──────┴─────────────┐
│ Return "NO"        │

       │

│    End     │

The function determines if two strings share a common substring by checking for any common character. It marks the characters of s1 in a boolean array and then iterates through s2, returning "YES" immediately if a match is found. If no match exists, it returns "NO". This approach is efficient with O(n + m) complexity.
