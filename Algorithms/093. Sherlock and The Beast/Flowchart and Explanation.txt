│ Read input n                       │
│ fives = n                          │
└──────┬─────────────────────────────┘
       │
┌──────┴─────────────────────────────┐
│ While fives % 3 != 0               │
│   fives -= 5                       │
└──────┬─────────────────────────────┘
       │
┌──────┴─────────────────────────────┐
│ If fives < 0                       │
│   Print "-1" and End               │
└──────┬─────────────────────────────┘
       │
┌──────┴─────────────────────────────┐
│ Print fives number of '5's         │
│ Print (n - fives) number of '0's   │
│ Print newline                      │
└──────┬─────────────────────────────┘
       │
┌──────┴─────┐
│    End     │
└────────────┘

The function generates the largest “Decent Number” consisting of digits 5 and 0 such that the number of 5s is divisible by 3 and the number of 0s divisible by 5. It starts with all digits as 5 and reduces in multiples of 5 until divisibility by 3 is achieved. If no valid combination exists, it prints -1. This ensures the largest possible decent number.
