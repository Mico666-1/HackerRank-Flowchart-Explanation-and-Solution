│    Start    │

       │
┌──────┴─────────────┐
│ Read array A[n]    │
└──────┬─────────────┘
       │
┌──────┴───────────────────────────┐
│ For i = 1 to n-1                 │
│   value = A[i]                   │
│   j = i - 1                      │
└──────┬───────────────────────────┘
       │
┌──────┴─────────────────────────────┐
│ While j ≥ 0 and A[j] > value       │
│   A[j+1] = A[j]                    │
│   j = j - 1                        │
└──────┬─────────────────────────────┘
       │
┌──────┴─────────────┐
│ A[j+1] = value     │
└──────┬─────────────┘
       │
┌──────┴─────────────┐
│ End For Loop       │
└──────┬─────────────┘
       │
┌──────┴─────────────┐
│ Print sorted array │
└──────┬─────────────┘
       │
┌──────┴───────┐
│     End      │
└──────────────┘

The program performs an insertion sort on the given array by iterating through its elements starting from the second one. For each element, it stores the value temporarily and compares it with the previous elements, shifting larger elements one position to the right to make space. When the correct position is found, it inserts the stored value. After the process completes, the array is fully sorted and printed as the output.
