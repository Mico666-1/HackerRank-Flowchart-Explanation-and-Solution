The function computes when each customer’s order will finish (orderTime + prepTime) and pairs it with the customer’s 1-based index. It then sorts the orders by finish time, breaking ties using the original order index. Finally, it returns the indices in the sequence the orders will be served, providing the correct service order efficiently.
